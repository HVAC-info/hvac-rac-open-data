### Integration layer (B544(E))

This repo includes an integration layer for **Control BOX B544(E)**:

- MODBUS register/coil map: `data/b544e_modbus_registers.csv`
- BACnet object list (MS/TP): `data/b544e_bacnet_objects.csv`
- Troubleshooting error codes: `data/b544e_troubleshooting_error_codes.csv`

See `docs/B544E_INTEGRATION.md` for key notes (serial settings, command throttling, device instance/MAC rules).
